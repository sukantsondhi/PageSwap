## ğŸ“– How to Use

1. **Upload Files** - Drag and drop PDFs or images onto the upload area, or click to browse
2. **View Pages** - All pages/images will be displayed as thumbnails with original and new position numbers
3. **Rearrange** - Drag and drop pages to reorder them as you wish
4. **Select & Delete** - Use checkboxes to select multiple pages, then delete unwanted ones
5. **Export** - Click "Export as PDF" to download your rearranged document

## ğŸ’» Local Development

1. Clone the repository:

   ```bash
   git clone https://github.com/sukantsondhi/PageSwap.git
   ```

2. Navigate to the project folder:

   ```bash

   ```
